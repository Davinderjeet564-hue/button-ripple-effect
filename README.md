# Button Ripple Effect

A lightweight, interactive button component that displays a ripple animation when clicked. Built with vanilla HTML, CSS, and JavaScript.

## 🎯 Overview

This project demonstrates a modern UI effect where clicking a button generates a faint ripple animation that spreads outward from the click point. It's a popular design pattern used in modern web applications to provide visual feedback to user interactions.

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and ripple animations
- **JavaScript** - Event handling and ripple effect logic

## ✨ Features

- Smooth ripple animation on button click
- Responsive and lightweight
- Easy to integrate into existing projects
- Cross-browser compatible
- Pure vanilla JavaScript (no dependencies)

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation & Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/Davinderjeet564-hue/button-ripple-effect.git
   cd button-ripple-effect
   ```

2. Open `index.html` in your web browser

3. Click the button to see the ripple effect in action!

## 💡 How It Works

1. When a button is clicked, the click position is detected
2. A ripple element is created at the click coordinates
3. CSS animation expands the ripple outward with a fade-out effect
4. The ripple element is removed after the animation completes

## 🎨 Customization

You can customize the ripple effect by modifying:
- **Animation duration** - Adjust the animation speed in CSS
- **Ripple color** - Change the background color of the ripple
- **Ripple size** - Modify the spread radius in JavaScript
- **Button styling** - Update button appearance in CSS
